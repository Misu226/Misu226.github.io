博客平台选择：

这里选择用Hexo框架建立静态博客。
原因是Hexo框架搭建主流，遇到错误，网上的解决方法多

系统：Win10 x64
工具：
node.js（LTS长期支持版本）
git
（安装步骤都非常简单，一直next，下一步就可以了，默认安装就行。）

##注意点
cmd打开检查是否安装成功
输入node -v，npm -v检查，如果出来版本号即为安装成功

输入npm的淘宝镜像

1.运行测试hexo s

打开浏览器，输入localhost:4000,看到
2.ctrl+c 退出
3.更改博客，vim，也可以直接记事本打开，或者下载markdown插件
win10直接写好然后放到blog\source_posts目录下就行了。
markendown语法。
vim进去后esc+:wq
4.hexo指令
https://hexo.io/zh-cn/docs/commands.html
5.
hexo clean
hexo g
